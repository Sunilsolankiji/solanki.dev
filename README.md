## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn  

### Installation

```bash
git clone https://github.com/SunilSolankiji/sunil-portfolio.git
cd sunil-portfolio
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
