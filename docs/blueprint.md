# **App Name**: Solanki.Dev

## Core Features:

- Hero Section: Hero section: prominent display of name, tagline, and a professional photo.
- About Section: About Section: Biography emphasizing a passion for coding, learning, and personal website creation.
- Projects Showcase: Projects Showcase: Visual display of GitHub repositories with descriptions, screenshots, and tech tags, linked to respective repos on GitHub.
- Skills Section: Skills & Technologies: List skills like HTML, CSS, TypeScript/JavaScript, Angular (v19) to highlight interests in full-stack development.
- Contact Section: Contact Section: Email link, social media icons (GitHub, LinkedIn:  https://in.linkedin.com/in/sunil-solanki-28b987175; https://github.com/SunilSolankiji).
- Responsiveness: Responsive layout for optimal viewing on different devices, including mobile-first design principles.
- Firebase Hosting: Integration with Firebase hosting for site deployment.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to convey calmness and reliability.
- Background color: Light grey (#F5F5F5) for a clean and modern backdrop.
- Accent color: Teal (#008080) to highlight interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and 'Inter' (sans-serif) for body text, for a clean, readable experience.
- Simple, line-based icons for skills, project categories, and contact information.
- Grid layout for the project showcase and a single-column layout for mobile devices to ensure content is easily accessible.
- Subtle scroll animations and transition effects to create a dynamic and engaging user experience.
